# Local Drop 🚀

A high-performance, open-source, cross-platform Progressive Web App (PWA) framework designed to securely transfer files directly over Local Area Networks (LAN) without internet dependency.

## Key Blueprint Highlights
- **OS Autopilot Mapping:** Auto-detects and structures paths smoothly across Windows (including OneDrive hooks), Linux, and macOS.
- **Accurate Wire Metrics:** Tracks actual wireless network over-the-air transit durations versus system disk write operations.
- **PWA Ready Architecture:** Zero browser address interface footprint when stored as a standalone homescreen utility tool.
